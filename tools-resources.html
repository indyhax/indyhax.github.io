CRYPTO TOOLS /
RESOURCES

A simple directory of useful crypto tools for CTF players. These tools help with decoding, hashes, ciphers, tokens, number conversions, and public-key crypto basics.

CRYPTO TOOLS

CyberChef

Data transforms

CyberChef transforms data in the browser. It can decode Base64 and hex, convert numbers, calculate hashes, work with XOR, and chain multiple steps together.

Open Tool

dCode

Cipher solvers

dCode has a large set of solvers for classic ciphers, encodings, math puzzles, and text transformations.

Open Tool

CaesarCipher.org

Letter shifts

CaesarCipher.org focuses on Caesar shifts. It is a simple way to see how rotating letters through the alphabet changes text.

Open Tool

CrackStation

Hash lookup

CrackStation checks hashes against lookup tables. It is best for common unsalted hashes where the original value may be a simple word or phrase.

Open Tool

jwt.io

JWT viewer

jwt.io decodes JSON Web Tokens so the header and payload are readable. It also shows the algorithm and signature section.

Open Tool

JWT Cracker

Weak secrets

JWT Cracker tests possible secrets for signed JWTs. It helps show why short or guessable token secrets are unsafe.

Open Tool

FactorDB

Number factors

FactorDB stores known integer factorizations. It can help identify whether a large number breaks into smaller prime factors.

Open Tool

dCode RSA

RSA calculator

dCode RSA works with common RSA values like moduli, public exponents, ciphertexts, and primes.

Open Tool

ASCII Converter

Text and numbers

ASCII converters translate between text and numeric forms such as decimal, hex, and binary.

Open Tool

Diffchecker

Compare text

Diffchecker compares two pieces of text and highlights changes. It is useful when two outputs look almost the same.

Open Tool

regex101

Pattern testing

regex101 helps test regular expressions and explains what each part does. It is useful for pulling patterns out of logs or text.

Open Tool

Google

Research

Search engines help look up file formats, error messages, cipher names, tool documentation, and unfamiliar CTF terms.

Open Tool
